# Creonix Demo Hub

Интерактивная демо-платформа ERP+CRM системы Creonix — показывает, как продукт выглядит и работает в разных отраслях бизнеса.

## Что это

Creonix Demo Hub — это набор из 10 отраслевых сценариев, каждый из которых демонстрирует управленческий интерфейс под конкретный бизнес: строительство, ритейл, швейное производство, бутик, агро, пекарня, ресторан, логистика, производство и дистрибуция.

Каждое демо включает:
- Главный экран с KPI, рисками и AI-сигналами
- CRM и операционный журнал
- Финансовый блок с графиками выручки и план-факт
- BI-аналитику с прогнозом
- AI-помощника с рекомендациями
- Реестр документов и блок до/после внедрения

## Отрасли

| Демо | Тема | Особенность |
|------|------|-------------|
| Строительство | Enterprise | Доска объектов, акты, бюджет |
| Продуктовый ритейл | Glass | Срок годности, полки, маржа категорий |
| Швейный цех | Glass | Партии, раскрой, ОТК, узкие места |
| Бутик | Glass | Коллекции, sell-through, клиентинг |
| Агро | Neo-dark | Поля, сезон, техника, прогноз |
| Пекарня | Glass | Смена, выпуск, свежесть |
| Ресторан | Glass | Загрузка, кухня, темп сервиса |
| Логистика | Enterprise | Маршруты, SLA, диспетчер |
| Производство | Neo-dark | OEE, линии, простои |
| Дистрибуция | Enterprise | Дебиторка, каналы, выручка |

## Технологии

- **Next.js 14** — App Router, серверные компоненты
- **React 18** + **TypeScript 5**
- **Tailwind CSS 3** — кастомные переменные под каждый бизнес
- **Recharts 2** — графики (Area, Bar, Line)
- **Framer Motion 11** — анимации
- **Lucide React** — иконки

## Запуск

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev -- --port 3001

# Сборка для продакшена
npm run build
npm start
```

Откройте [http://localhost:3001](http://localhost:3001).

## Структура проекта

```
creonix/
├── app/
│   ├── page.tsx              # Главная страница с выбором демо
│   └── demo/[business]/      # Динамические страницы демо
├── components/
│   ├── charts/               # Recharts компоненты
│   ├── layout/               # Три темы: glass, enterprise, neo-dark
│   ├── tables/               # CRM, операции, инвентарь
│   ├── ui/                   # Badge, Button, Card, Tabs
│   └── widgets/              # Отраслевые виджеты (gauge, heatmap, etc.)
├── data/
│   ├── businesses.ts         # Профили 10 бизнесов
│   └── demoData.ts           # Демо-данные по каждому сценарию
├── lib/
│   ├── types.ts              # TypeScript типы
│   ├── utils.ts              # cn(), форматтеры, классы
│   └── demo.ts               # Масштабирование данных по периодам
└── public/
    └── logo.png              # Логотип Creonix
```

## UI темы

- **Glass** — светлый, полупрозрачный стиль для сервисного бизнеса
- **Enterprise** — строгий корпоративный стиль для B2B
- **Neo-dark** — тёмная консольная тема для производства и агро
